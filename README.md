# YuHang-Skill 🧬

> 我把自己蒸馏成了一个 Skill。

## 这是什么？

这是我的数字分身——包含了我的个人档案、决策模型、核心价值观和技术能力矩阵。

如果你想让 AI 更了解我，或者在需要做决策时想参考我的思维框架，可以直接调用这个 Skill。

## 包含内容

- 📋 **个人档案** - 基本信息、求学经历、人生经历
- 🧠 **决策模型** - Claw Decision Model v1.0（非对称风险、10-10-10法则、门类型判断）
- 💪 **技术能力矩阵** - 各领域掌握程度和项目经验
- 🎯 **核心价值观** - 独立自主 + 实用主义

## 怎么用？

把 `SKILL.md` 放到你的 Claude Code Skills 目录，然后在对话中提到：

- "按照 YuHang 的决策模型分析一下..."
- "结合 YuHang 的价值观给个建议..."
- "参考 YuHang 的技术能力..."

AI 就会基于我的思维方式来回应你。
