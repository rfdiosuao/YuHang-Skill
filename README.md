### 🧠 Claw Decision Model v1.0

**四象限风险定位系统**

```
        收益上限高
              ↑
    非对称选择 │ 对称选择（赌徒陷阱）
              │
──────────────┼──────────────→ 收益上限低
              │
    别碰       │ 非对称选择
              ↓
        损失下限高
```

**核心原则**:
- ✅ 追求**非对称风险**：损失有限，收益无限
- ❌ 避开**赌徒陷阱**：收益有限，损失无限
- 🚪 **别想太多，先推门**
