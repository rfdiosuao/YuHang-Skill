## 决策模型 (Claw Decision Model v1.0)

### 四象限坐标定位

```
        收益上限高
              ↑
    非对称选择 │ 对称选择（赌徒陷阱）
              │
──────────────┼──────────────→ 收益上限低
              │
    别碰       │ 非对称选择
              ↓
        损失下限高
```

**核心原则**:
- 追求**非对称风险**：损失有限，收益无限
- 避开**赌徒陷阱**：收益有限，损失无限
- **别想太多，先推门**

### 决策表格模板

```
📌 决策事项：______________

1️⃣ 10-10-10 测试结果：
   - 10天后：______________
   - 10个月后：____________
   - 10年后：______________

2️⃣ 门类型：□ 双向门  □ 单向门

3️⃣ 风险对称性：
   - 最坏情况：____________（能承受吗？□）
   - 最好情况：____________（有天花板吗？□）

4️⃣ 信息量：□<30%  □30-70%  □>70%

5️⃣ 退出条件：
   如果 ____ 个月内没有达成 ____，我就退出。

✅ 结论：□ 立刻行动  □ 再想想  □ 放弃
```

### 决策要点

1. **10-10-10法则**: 用三个时间维度审视决策
2. **门类型判断**: 
   - 双向门 = 可逆决策，快速尝试
   - 单向门 = 不可逆决策，慎重思考
3. **风险对称性**: 永远追求非对称收益
4. **信息量评估**: 信息不足时，用小成本试错
5. **退出机制**: 设定明确的退出条件，没达成预期果断退场
